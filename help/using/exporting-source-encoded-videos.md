# Exportar vídeos de origem e codificados{#exporting-source-and-encoded-videos}

No painel Procurar, os vídeos de origem são rotulados como *Vídeo principal*. Os vídeos codificados são rotulados como *Vídeo*. Você pode exportar vídeos e vídeos principais. Para exportar arquivos de vídeo, selecione-os no painel Procurar e vá para **[!UICONTROL File]** > **[!UICONTROL Export]**.

Consulte [Exportar ativos do Adobe Dynamic Media Classic](exporting-assets-from-dmc.md#exporting-assets-from-dmc).
