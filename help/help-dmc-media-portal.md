# Ajuda do Dynamic Media Classic Media Portal{#help-for-scene-media-portal}

O seguinte documento de Ajuda também está disponível no Dynamic Media Classic:

**Use o Dynamic Media Classic Media Portal**  - O Adobe Dynamic Media Classic Media Portal é um ambiente baseado em navegador que fornece aos usuários do portal acesso fácil para fazer upload, navegar, pesquisar, visualizar e exportar ativos em formatos aprovados pela empresa.
